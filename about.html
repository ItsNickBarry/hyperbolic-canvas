Hyperbolic Geometry

Hyperbolic Canvas is a Javscript library which uses the Poincaré disk model of the hyperbolic plane to map hyperbolic space onto an HTML canvas.

Hyperbolic Canvas was created by Nick Barry, with thanks to:

  • x13machine, for various debugging
  • Authors of Skeleton, for the site's CSS framework
  • János Bolyai
